## 📖 Project Overview
This repository contains my Digital Literacy project for the CSE0001 course. The purpose of this project is to understand and apply important digital skills that are useful for students in today’s digital world.

The project is divided into five tasks, each focusing on a different aspect of digital literacy. Through these tasks, I explored various tools, platforms, and practices that help in improving online presence, communication, and online safety.

---

## 📂 Tasks Included

### 🟢 Task 1 – Digital Literacy Infographic
Created an infographic explaining digital literacy, useful tools, and safe internet practices using Canva.

### 🟢 Task 2 – Digital Portfolio
Set up professional profiles on platforms like GitHub and LinkedIn to build an online presence.

### 🟢 Task 3 – Platforms
Explored coding platforms and created a Google Form for a digital literacy quiz.

### 🟢 Task 4 – Email Etiquette
Wrote professional emails and created a checklist for responsible social media use.

### 🟢 Task 5 – Cybercrime Awareness
Prepared a case study on cybercrime and a prevention checklist for staying safe online.
